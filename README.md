# Local API Key Vault

A secure, client-side application for managing API keys with local encryption. This project demonstrates modern web development practices including client-side encryption, responsive design, and clean code architecture.

## Features

- 🔒 Client-side encryption using Web Crypto API
- 📱 Responsive design that works on all devices
- 💾 Local storage with secure encryption
- 🎨 Modern UI with smooth animations
- 🔄 Real-time updates

## Technologies Used

- HTML5, CSS3, JavaScript (ES6+)
- Tailwind CSS for styling
- Web Crypto API for encryption
- Particles.js for background effects

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/api-key-vault-portfolio.git
   cd api-key-vault-portfolio
   ```

2. Open `index.html` in your browser (no build step required)

## Security

- All data is encrypted before being stored in localStorage
- Uses AES-GCM encryption with PBKDF2 key derivation
- No data is ever sent to any server - everything stays in your browser
